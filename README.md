# SQL Analysis Portfolio

## Overview
This repository contains my SQL data projects focused on data exploration, data cleaning, and dashboard preparation.

The projects in this repository demonstrate practical SQL skills such as:
- data cleaning and transformation
- aggregations and filtering
- joins
- Common Table Expressions (CTEs)
- window functions
- temporary tables
- views
- preparing data for visualization

## Projects

### 1. Global COVID-19 SQL Analysis and Tableau Dashboard
This project analyses global COVID-19 case, death, and vaccination data using SQL Server and presents key insights in Tableau.

**Main skills used:**
- joins
- aggregations
- window functions
- CTEs
- temp tables
- views
- Tableau dashboard preparation

Folder: `Covid data exploration project`

---

### 2. Nashville Housing Data Cleaning in SQL
This project cleans and transforms the Nashville Housing dataset using SQL Server by standardizing dates, filling missing values, splitting address columns, removing duplicates, and dropping unnecessary columns.

**Main skills used:**
- data cleaning
- `UPDATE`
- `ALTER TABLE`
- `ISNULL()`
- `SUBSTRING()`
- `PARSENAME()`
- `CASE WHEN`
- `ROW_NUMBER()`

Folder: `Data Cleaning project`
